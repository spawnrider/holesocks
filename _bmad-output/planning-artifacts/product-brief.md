# Product Brief — HoleSocks

**Date** : Mai 2026  
**Scope** : Cas d'usage démonstration BMAD — E-commerce satirique

---

## Executive Summary

**HoleSocks** est une e-commerce B2C qui vend des chaussettes volontairement trouées, en assumant l'absurde avec second degré. Le produit démontre un workflow agentique complet (BMAD) : conception produit → architecture → implémentation → QA, sur un cas assez simple pour être livrable en démo salon, mais assez riche en interactions pour illustrer comment les agents collaborent.

**Audience cible** : Toute personne ayant un jour regardé une chaussette trouée le lundi matin.  
**Ton** : Absurde assumé, légèrement philosophique, jamais agressif.  
**Baseline** : "L'idée la plus absurde possible, exprès."

---

## Le Concept

Les chaussettes finissent *toujours* trouées. Les clients le savent. Les fabricants le savent. La question n'est pas "comment l'éviter ?" mais "pourquoi pas le célébrer ?"

HoleSocks ne vend pas des chaussettes abîmées. Elle vend de l'**avant-garde confortable** — une provocation douce, du design intentionnel, et surtout : de l'humour.

### Positionnement

- **Pas un défaut** : une *feature*
- **Pas un rejet de la qualité** : une revendication d'authenticité
- **Pas une arnaque** : une promesse tenue avec le sourire

## Catalogue & Offre

### Trois niveaux de perforation

| Niveau | Nom | Description |
|--------|-----|-------------|
| 1 | **Léger** | Un discret frisson d'air — "On appelle ça la respiration, pas un trou." |
| 2 | **Aéré** | Ventilation assumée — Idéal pour les chaudes journées de déni. |
| 3 | **Catastrophe** | Le trou qui questionne vos choix de vie — Livré avec un mot de soutien. |

Chaque produit inclut :
- Un nom de modèle absurde (ex. "Le Philosophe", "L'Existentiel", "Le Lundi Noir")
- Une description produit décalée et légèrement philosophique
- Une photo produit mettant en valeur le(s) trou(s) comme argument de vente

### Cross-sell : Le Kit du Bricoleur

"Agrandissez vous-même vos trous" — kit vendu séparément.  
Contenu : pointeau, guide des perforations tendance, patch moral.

## Fonctionnalités Clés

### Navigation & Filtrage
- Filtrer par niveau de trou (Léger / Aéré / Catastrophe)
- Filtrer par couleur et taille (38–40, 41–43, 44–46)
- Filtrer par style de ventilation (orteil exposé, talon aéré, semelle ouverte)

### Fiche Produit
- Nom du modèle + accroche absurde
- Niveau de trou avec jauge visuelle
- Description rédigée avec le ton de la marque
- Avis clients (ton absurde intentionnel)

### Panier & Checkout
- Ajout au panier classique
- Checkout simple, sans compte client
- Message de confirmation décalé

### Avis Clients
- Note + citation courte
- Ton volontairement absurde et enthousiaste
