# Components Directory

Organized by feature. Each subdirectory contains related React components.

## Structure

- **filters/** - Filter bar and filter chip components for catalogue page filtering
- **product/** - Product card, hole gauge, and product-related components
- **reviews/** - Review list, review card, and review form components
- **kit/** - Cross-sell bricoleur kit component
- **layout/** - Header, footer, and layout wrapper components
- **ui/** - Reusable UI primitives (buttons, inputs, etc.)

## Naming Conventions

- PascalCase for component files: `ProductCard.tsx`
- camelCase for utility functions/hooks: `useFilters.ts`
- One component per file
